# AI Test Portal

[![Frontend](https://img.shields.io/badge/Frontend-React-blue)](#tech-stack)
[![Backend](https://img.shields.io/badge/Backend-Node.js-green)](#tech-stack)

**AI Test Portal** is a full-stack web application that allows users to generate and take quizzes using AI. This project is designed to practice real-world frontend and backend development skills.

---

##  Features

- **User Signup & Login** – Secure authentication for users.  
- **AI-based Quiz Generation** – Generate quizzes automatically using AI.  
- **Quiz Setup and Attempt** – Configure quizzes and attempt them interactively.  
- **Quiz Result and Review** – View results and review answers after completing a quiz.  
- **User Profile Page** – Manage and view user information and quiz history.  

---

##  Tech Stack

### Frontend
- **React.js** – Dynamic and responsive UI   
- **Axios** – API requests to backend  

### Backend
- **Node.js & Express.js** – Server-side logic and REST APIs  
- **MongoDB** – NoSQL database for storing users, quizzes, and results
